B11.2 Language focus: Quantities
Some and any
We use
● some in positive sentences: I’ve got some money.
We can use
● some in polite questions when we are offering or asking for something: Would you
like some water? Can I have some tomatoes, please?
We use
● any in negative sentences: I haven’t got any paper.
We also use
● any in questions to find out if something is or isn’t available: Have you caught any
fish today?
