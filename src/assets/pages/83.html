B4.8 Language focus: If sentences
We use
● if sentences to talk about something that depends on something else.
We use
● if sentences and will to talk about future possibilities, for example:
If I eat healthy food now, I will be healthier when I am older.
We use
● if sentences in the present to talk about habits or general truths, for example:
If I wake up early, I eat breakfast before going to school.
Note:
The two parts of
● if sentences can go either first or second in the sentence, for example:
If you eat too many cakes, you will get fat.
You will get fat if you eat too many cakes.
The same sequence of tenses can be used for sentences introduced by
● provided, as long as, unless
and will.
